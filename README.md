## Installation

```bash
cd ./scripts

./start.sh # Créer et démarre le conteneur
```

## Usage

```bash
cd ./scripts

./kill.sh # Supprime le conteneur ainsi que les volumes associés

./stop.sh # Stop le conteneur
```
